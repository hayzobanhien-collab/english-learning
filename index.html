Cấu trúc đảo ngữ với "No sooner ... than"

Diễn tả một hành động vừa mới xảy ra thì hành động khác xen vào ngay lập tức (Vừa mới... thì...).

No sooner + Had + S + V3/ed + than + S + V2/ed

Ví dụ: No sooner had we implemented the code than the server crashed.

Cấu trúc giả định với "It is crucial that"

Dùng để nhấn mạnh sự quan trọng, cấp bách cực kỳ của một sự việc nào đó.

It is crucial that + S + V (nguyên thể không chia)

Ví dụ: It is crucial that he master object-oriented programming fundamentals.
